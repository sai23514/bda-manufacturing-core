## 🚀 How to Seed the Database

### Step 1: Make Sure MongoDB is Running

```bash
# Check if MongoDB is running
mongosh

# If not running, start it:
# macOS
brew services start mongodb-community

# Linux
sudo systemctl start mongodb

# Windows
# Start MongoDB service from Services panel
```

### Step 2: Run the Seeder

```bash
# Make sure you're in the server directory
cd server

# Run the seed script
npm run seed
```

## 🔄 Re-seeding the Database

**Warning:** This will **DELETE ALL EXISTING DATA** and replace it with sample data.

```bash
npm run seed
```

This is useful when you want to:
- Reset to clean sample data
- Clear test data you've created
- Start fresh for a demo

## 🛠️ Customizing Sample Data

To modify the sample data, edit:
```
server/src/seeders/seedData.js
```

You can:
- Add more users
- Change lead details
- Modify company names
- Adjust values and statuses
- Add more activities

Then run `npm run seed` again.
